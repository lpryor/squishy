AsyncQueue

trait AsyncQueue[M] extends Queue[M]

A wrapper around an asynchronous SNS queue that supports idiomatic Scala.

Note: implementations will transparently acquire and cache this queue's URL when required. See the note about queue URLs in the class description of squishy.Queue.

M

The type of message that can be sent to and received from this queue.
