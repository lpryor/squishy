squishy

package squishy

Squishy provides an idiomatic Scala API for the Amazon Simple Queue Service (SQS).

Squishy provides two views on a SQS queue, SyncQueue and AsyncQueue, that expose the synchronous and asynchronous SQS APIs respectively. Views are created by extending one or both of the aforementioned traits:

object MySyncQueue extends SyncQueue[String] {
  ...
}

object MyAsyncQueue extends AsyncQueue[String] {
  ...
}

object MySuperQueue extends SyncQueue[String] with AsyncQueue[String] {
  ...
}

See Queue for information on the configuration parameters accepted by both queue types. See SyncQueue or AsyncQueue for information on how to use those interfaces.
